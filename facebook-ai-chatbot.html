📄 Step 1 — Your bot server code (Node.js)

Click "Generate Code" to create a personalized server file with your config already baked in.

Save this as index.js on your computer.

📦 Step 2 — package.json

Save this as package.json in the same folder.

🚀 Step 3 — Deploy free on Render.com
1

Create a GitHub repo

Go to github.com → New repo → Upload your index.js and package.json

2

Deploy on Render (free)

Go to render.com → New → Web Service → Connect your GitHub repo → Build: npm install, Start: node index.js. Choose the free tier.

3

Add environment variables on Render

In your Render service → Environment → Add:

GEMINI_API_KEY = your-key-here FB_PAGE_TOKEN = your-facebook-page-token FB_VERIFY_TOKEN = any-secret-string-you-choose
4

Connect to your Facebook Page

Go to developers.facebook.com → Your App → Messenger → Webhooks → Add your Render URL as: https://your-app.onrender.com/webhook. Use your FB_VERIFY_TOKEN for verification. Subscribe to messages.

⚠️ On Render's free tier, the server sleeps after 15 min of inactivity. Set up a free uptime monitor at uptimerobot.com to ping your URL every 5 minutes — this keeps it awake 24/7 for free.
🔑 Getting Your Anthropic API Key (Free)
  1. Go to console.anthropic.com
  2. Sign up for a free account (no credit card needed to start)
  3. Go to API Keys → Create Key
  4. Copy the key and paste it in the Setup tab above
  5. Anthropic gives free credits to new accounts — enough to test and launch
📘 Getting Your Facebook Page Token
  1. Go to developers.facebook.com and log in
  2. Click My Apps → Create App → choose Business
  3. Add the Messenger product to your app
  4. Under Messenger settings → Access Tokens → select your Facebook Page
  5. Click Generate Token and copy it — this is your FB_PAGE_TOKEN

🛠 Troubleshooting
Bot not replying? Check that your Render service is running and the webhook is verified on Facebook Developers dashboard.
Webhook verification fails? Make sure FB_VERIFY_TOKEN in Render matches exactly what you entered in Facebook's webhook settings.
Server sleeping? Set up UptimeRobot to ping your URL every 5 minutes — free at uptimerobot.com.
Wrong language? Update the language field in Setup and regenerate your code.
